对应 wasm 的 memory.size 指令

调用此函数时，它将以页为单位返回当前的内存大小。 当前的 WebAssembly 页面大小为 65536 字节 (64 KB)。

参数 MEM 是返回内存大小的数字索引。 请注意，当前 WebAssembly 规范仅支持一个内存，因此要求传入零。

该参数与 future WebAssembly 修订版向前兼容。 如果将非零参数传递给此函数，则它将当前无条件终止。
