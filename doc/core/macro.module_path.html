Macro core::module_path1.0.0[−][src]

macro_rules! module_path {
    () => { ... };
}
Expand description

扩展为代表当前模块路径的字符串。

当前模块路径可以被认为是引回到 crate root 的模块层次结构。 返回路径的第一部分是当前正在编译的 crate 的名称。

Examples

mod test {
    pub fn foo() {
        assert!(module_path!().ends_with("test"));
    }
}

test::foo();
