Keyword true[−][src]

Expand description

bool 类型的值，表示逻辑 true。

逻辑上 true 不等于 false。

检查 true 的控制结构

Rust 的几个控制结构将检查 bool 条件是否评估为 true。

  • if 表达式中的条件必须为 bool 类型。 只要该条件评估为 true，if 表达式就会采用第一个块的值。 但是，如果条件计算结果为 false，则表达式为 else 块的值 (如果有)。

  • while 是另一个控制流构造，期望使用 bool 类型的条件。 只要条件评估为 true，while 循环将连续评估其关联的块。

  • match 分支上可以有保护子句。
