Macro std::println1.0.0[−][src]

macro_rules! println {
    () => { ... };
    ($($arg:tt)*) => { ... };
}
Expand description

用换行符打印到标准输出。

在所有平台上，换行符仅是 LINE FEED 字符 (\n/U+000A) (没有其他的 CARRIAGE RETURN (\r/U+000D))。

使用 format! 语法将数据写入标准输出。 有关更多信息，请参见 std::fmt。

println! 仅用于程序的主要输出。请改用 eprintln! 打印错误和进度消息。

Panics

如果写入 io::stdout 失败，则为 Panics。

Examples

println!(); // 只打印换行符
println!("hello there!");
println!("format {} arguments", "some");
