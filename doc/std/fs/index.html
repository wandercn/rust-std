Module std::fs1.0.0[−][src]

Expand description

文件系统操纵操作。

该模块包含一些操作本地文件系统内容的基本方法。 该模块中的所有方法均表示跨平台文件系统操作。 在 std::os::$platform 的扩展名 traits 中可以找到特定于平台的其他功能。
