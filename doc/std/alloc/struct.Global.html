pub struct Global;
🔬 This is a nightly-only experimental API. (allocator_api #32838)
Expand description

内存分配器。

此类型通过将调用转发到用 #[global_allocator] 属性注册的分配器 (如果有的话) 或 std crate 的默认值来实现 Allocator trait。

Note: 尽管此类型不稳定，但是可以通过 free functions in alloc 访问其提供的功能。
