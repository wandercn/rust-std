Expand description

提取指针的元数据组件。

*mut T，&T 或 &mut T 类型的值可以隐式强制转换为 * const T，因此可以直接传递给此函数。

Example

#![feature(ptr_metadata)]

assert_eq!(std::ptr::metadata("foo"), 3_usize);
