🔬 This is a nightly-only experimental API. (wasi_ext #71213)

将自定义 dirflags 参数传递给 path_open。

此选项将 dirflags 参数配置为 OpenOptions 最终将调用的 path_open 系统调用。 dirflags 参数配置如何查找文件，当前主要影响是否遵循符号链接。

默认情况下，此值为 __WASI_LOOKUP_SYMLINK_FOLLOW，否则将遵循符号链接。 您可以使用 0 调用此方法以禁用以下符号链接
