unsafe fn from_raw_handle(handle: RawHandle) -> Self

[src]

从指定的原始句柄创建一个新的 I/O 对象。

此函数将消耗给定句柄的所有权，将关闭句柄的责任传递给返回的对象。

此函数也是不安全的，因为当前返回的原语具有它们是包装的文件描述符的唯一所有者的约定。 使用此函数可能会意外地违反该协定，这可能导致依赖于该协定为真的代码中的内存不安全。
