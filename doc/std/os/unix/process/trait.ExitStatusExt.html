Expand description

process::ExitStatus 和 ExitStatusError 的 Unix 特定扩展。

在 Unix 上，ExitStatus不一定表示退出状态，它已传递给 exit 系统调用或由 ExitStatus::code() 返回。 它表示由 wait 系列系统调用之一返回的任何等待状态。

Unix 等待状态 (Rust ExitStatus) 可以代表 Unix 退出状态，但也可以代表其他类型的进程事件。

trait 是密封的: 不能在标准库之外实现。 这是为了将来的附加方法不会破坏更改。

从 wait 的原始底层整数状态值创建一个新的 ExitStatus 或 ExitStatusError

该值应该是等待状态，而不是退出状态。

Panics

尝试从 0 的等待状态生成 ExitStatusError 时出现 panic。

创建 ExitStatus 总是会成功的，并且从不 panics。
