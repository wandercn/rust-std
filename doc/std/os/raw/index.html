特定于平台的类型，由 C 定义。

通过 FFI 进行交互的代码几乎肯定会使用 C 提供的基本类型，而这些基本类型的定义不如 Rust 的基本类型好。 该模块提供的类型将与 C 定义的类型匹配，以便与 C 交互的代码将引用正确的类型。
