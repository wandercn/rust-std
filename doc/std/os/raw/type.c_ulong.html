Type Definition std::os::raw::c_ulong1.1.0[−][src]

type c_ulong = u32;
Expand description

等效于 C 的 unsigned long 类型。

此类型将始终为 u32 或 u64。最值得注意的是，许多基于 Linux 的系统都使用 u64，但 Windows 则使用 u32。C 标准从技术上仅要求此类型是 long 大小的无符号整数，尽管在实践中，没有系统会使用既不是 u32 也不是 u64 的 ulong。
