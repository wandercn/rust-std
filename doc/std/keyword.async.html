Keyword async[−][src]

Expand description

返回 Future，而不是阻塞当前线程。

使用 fn，closure 或 block 前面的 async 将标记的代码转换为 Future。 因此，该代码将不会立即运行，而只会在返回的 future 为.await 时进行评估。

我们已经编写了 async book，其中详细介绍了 async/await 和与使用线程相比的取舍。

Editions

async 是从 2018 年版开始的关键字。

从 1.39 版本开始，它可以在稳定的 rust 中使用。
