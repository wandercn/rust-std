Primitive Type unit1.0.0[−]

Expand description

() 类型，也称为 “unit”。

() 类型只有一个值 ()，在没有其他有意义的值可以返回时使用。 () 最常见的隐含形式: 没有 -> ... 的函数隐含具有返回类型 ()，即，它们是等效的:

fn long() -> () {}

fn short() {}
Run

分号 ; 可用于在块末尾丢弃表达式的结果，从而使表达式 (从而使该块) 的值为 ()。

例如，

fn returns_i64() -> i64 {
    1i64
}
fn returns_unit() {
    1i64;
}

let is_i64 = {
    returns_i64()
};
let is_unit = {
    returns_i64();
};
