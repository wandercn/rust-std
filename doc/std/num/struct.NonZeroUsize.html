Struct std::num::NonZeroUsize1.28.0[−][src]

#[repr(transparent)]
pub struct NonZeroUsize(_);
Expand description

已知不等于零的整数。

这样可以进行一些内存布局优化。 For example, Option<NonZeroUsize> is the same size as usize:

use std::mem::size_of;
assert_eq!(size_of::<Option<core::num::NonZeroUsize>>(), size_of::<usize>());
