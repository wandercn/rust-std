Expand description

Clone 到借用数据的一般化。

某些类型通常可以通过实现 Clone trait 从借用变为拥有。 但是 Clone 仅适用于从 &T 到 T 的情况。 ToOwned trait 泛化 Clone 来构造给定类型的任何借用数据。

通常通过克隆从借用数据中创建拥有的数据。

Examples

let s: &str = "a";
let ss: String = s.to_owned();

let v: &[i32] = &[1, 2];
