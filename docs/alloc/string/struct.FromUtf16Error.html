pub struct FromUtf16Error(_);
Expand description

从 UTF-16 字节切片转换 String 时可能的错误值。

该类型是 String 上 from_utf16 方法的错误类型。

Examples

基本用法:

// 𝄞mu<invalid>ic
let v = &[0xD834, 0xDD1E, 0x006d, 0x0075,
          0xD800, 0x0069, 0x0063];

assert!(String::from_utf16(v).is_err());
