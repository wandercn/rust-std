Expand description

trait，用于在 main 函数中实现任意返回类型。

C-main 函数仅支持返回整数作为返回类型。 因此，实现 Termination trait 的每种类型都必须转换为整数。

默认实现将返回 libc::EXIT_SUCCESS 以指示成功执行。 如果发生故障，则返回 libc::EXIT_FAILURE。
