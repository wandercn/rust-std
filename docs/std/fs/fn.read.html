pub fn read<P: AsRef<Path>>(path: P) -> Result<Vec<u8>>
Expand description

将文件的全部内容读取为字节 vector。

这是使用 File::open 和 read_to_end 且导入次数较少且没有中间变量的便捷函数。 它会根据文件大小 (如果可用) 预分配缓冲区，因此通常比读入 Vec::new() 创建的 vector 更快。

Errors

如果 path 还不存在，则此函数将返回错误。 根据 OpenOptions::open，可能还会返回其他错误。

如果在读取 io::ErrorKind::Interrupted 以外的其他类型的错误时遇到错误，它也会返回错误。

Examples

use std::fs;
use std::net::SocketAddr;

fn main() -> Result<(), Box<dyn std::error::Error + 'static>> {
    let foo: SocketAddr = String::from_utf8_lossy(&fs::read("address.txt")?).parse()?;
    Ok(())
}
