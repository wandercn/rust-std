Enum std::path::Component1.0.0[−][src]

pub enum Component<'a> {
    Prefix(PrefixComponent<'a>),
    RootDir,
    CurDir,
    ParentDir,
    Normal(&'a OsStr),
}
Expand description

路径的单个组成部分。

Component 大致对应于路径分隔符 (/ 或 \) 之间的子字符串。

该 enum 是通过迭代 Components 来创建的，而 Components 又是通过 Path 上的 components 方法创建的。

Examples

use std::path::{Component, Path};

let path = Path::new("/tmp/foo/bar.txt");
let components = path.components().collect::<Vec<_>>();
assert_eq!(&components, &[
    Component::RootDir,
    Component::Normal("tmp".as_ref()),
    Component::Normal("foo".as_ref()),
    Component::Normal("bar.txt".as_ref()),
]);
Run

Variants

Prefix(PrefixComponent<'a>)

Windows 路径前缀，例如 C: 或 \\server\share。

前缀类型种类繁多，有关更多信息，请参见 Prefix 的文档。

在 Unix 上不会发生。

RootDir

根目录组件出现在任何前缀之后和其他任何内容之前。

它代表一个分隔符，它指定路径从根开始。

CurDir

对当前目录的引用，即 ..

ParentDir

对父目录的引用，即 ...

Normal(&'a OsStr)

正常组件，例如 a/b 中的 a 和 b。

这个成员是最常见的一个，它代表对文件或目录的引用。
