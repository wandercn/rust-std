Expand description

SipHash 2-4 的实现。

See: https://131002.net/siphash/

SipHash 是通用的哈希函数: 它以良好的速度运行 (与 Spooky 和 City 竞争)，并允许强大的 keyed 哈希。

这使您可以从强大的 RNG (例如 rand::os::OsRng) 中键入哈希表。

尽管 SipHash 算法通常被认为是强大的，但它并非旨在用于加密目的。 这样，此实现的所有加密用途都是 strongly discouraged。
