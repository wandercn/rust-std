Struct core::cmp::Reverse1.19.0[−][src]

#[repr(transparent)]
pub struct Reverse<T>(pub T);
Expand description

用于逆序排序的辅助结构体。

该结构体是与 Vec::sort_by_key 等函数一起使用的助手，并且可以用于对键的一部分进行反向排序。

Examples

use std::cmp::Reverse;

let mut v = vec![1, 2, 3, 4, 5, 6];
v.sort_by_key(|&num| (num > 3, Reverse(num)));
assert_eq!(v, vec![3, 2, 1, 6, 5, 4]);
