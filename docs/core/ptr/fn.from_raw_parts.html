Expand description

根据数据地址和元数据形成 (possibly-wide) 裸指针。

此函数是安全的，但是返回的指针对于解引用并不一定是安全的。 对于切片，请参见 slice::from_raw_parts 的文档以了解安全要求。 对于 trait 对象，元数据必须来自指向相同底层 erased 类型的指针。
