Expand description

用于可变借用数据的 trait。

作为 Borrow<T> 的伴侣，此 trait 通过提供变量引用允许类型借用作为基础类型。 请参见 Borrow<T>，以了解更多有关借用另一种类型的信息。

从拥有的值中借用。

Examples

use std::borrow::BorrowMut;

fn check<T: BorrowMut<[i32]>>(mut v: T) {
    assert_eq!(&mut [1, 2, 3], v.borrow_mut());
}

let v = vec![1, 2, 3];

check(v);
